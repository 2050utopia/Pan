# Pan Sample: Github Contributors

一个简单的Sample，包含List页和Detail页。共由三个ViewModel部件完成：

1. UserViewModel：展示用户头像和名字，在List中和Detail中被重用
2. UserListViewModel：列表展示用户
3. UserDetailViewModel：展示用户详情，包括Push记录
